PDF Master

PDF Master is a straightforward npm package designed for generating static and dynamic PDF files in Node.js. It offers a seamless conversion of HTML templates (with the .hbs extension) into PDF documents.

Key features of PDF Master include:
  • Generate PDF files from HTML templates.
  • Create dynamic PDFs by incorporating data into templates.
  • Customize PDFs with custom headers and footers.

The package is available under the MIT License.

Covid-19 Tracker

This is a web app that provides live tracking of Coronavirus cases worldwide and in India. The app utilizes HTML, CSS, JavaScript, Materialize framework, Fetch API, and SVG for its development. Various APIs are used to retrieve data, including World Data, India Data, State Data, and Coronavirus News.

The project incorporates the Fetch API to obtain real-time information on active cases, deaths, and recoveries for the world, India, and Indian states. Additionally, the app features live news updates related to COVID-19.
